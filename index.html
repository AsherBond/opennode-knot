Welcome to OpenNode Knot

OMS Knot is cloud management solution built on top of the OMS core framework (http://opennode.github.com/opennode-management/)

Quick Installation

$ curl -s http://opennodecloud.com/download/oms-core-installer.sh | sh
$ cd oms
$ bin/plugin install opennode.oms.knot

Further documentation available at:

  • http://opennodecloud.com/docs/opennode.oms.core/
  • http://opennodecloud.com/docs/opennode.knot/
